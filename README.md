# Build a Game in HTML CSS & JavaScript

Um projeto simples de **jogo web** usando HTML, CSS e JavaScript.

## ✨ Sobre

Este projeto demonstra como criar um **jogo básico interativo**, ideal para praticar lógica de programação e manipulação do DOM.

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript

## 🎯 Funcionalidades

- Jogo interativo
- Lógica com JavaScript
- Interface simples e divertida

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
